# Migration guide

This document records breaking changes between major versions of
`@pyrx/synapse-react-native`. Patch and minor releases do not break
the public API; consult the [CHANGELOG](../CHANGELOG.md) for additive
changes.

## 0.1.0 → (future)

No breaking changes yet — this is the initial release. Migration
notes will appear here when a `0.2.x` or `1.0.0` release introduces
incompatible changes.

If you are integrating PYRX Synapse for the first time, start with the
[README](../README.md). If you are migrating from a direct integration
of the native iOS / Android SDKs (without this RN wrapper), see
[`MIGRATING-FROM-NATIVE.md`](./MIGRATING-FROM-NATIVE.md).
